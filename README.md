# skill.skill

Automatic skill farming for Codex.

## Use When
- You want a batch of fresh, creative skills.
- You want each skill to extend or specialize model behavior.
- You want one worker per approved idea and a concise root shelf.

## Quick Start
1. Generate seeds with `scripts/generate_seed_pack.py`.
2. Turn seeds into candidate ideas.
3. Validate the selected batch.
4. Dispatch one worker per approved skill.
5. Update `developed-skills.json` and rebuild `developed-skills.md`.

## Why Try It
It treats skills like a growing workshop: random seeds create surprising ideas, strict filters reject generic prompts, and the shelf stays short enough to browse.
